HTML Form Reference Cheat Sheet

Form element's

Tag Name Description
<input> input The <input> tag specifies an input field where the user can enter data.
<textarea> textarea The <textarea> tag defines a multi-line text input control.The size of a text area is specified by the cols and rows attributes.
<button> button The <button> element defines a clickable button.
<select> select The <select> element is used to create a drop-down list. & is most often used in a form, to collect user input.
<option> option The <option> tag defines an option in a select list.
<label> label The HTML <label> element provides identification for a specific <input> based on matching values of the <input>‘s id attribute and the <label>‘s for attribute.

Form Attribute's

Attribute Name Description
name name In order for a form to send data, it needs to be able to put it into key-value. This is achieved by setting the name attribute of the input element.
required required In HTML, input fields have an attribute called required which specifies that the field must include a value.
